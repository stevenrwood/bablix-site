Frequently Asked Questions

Do I need a Claude API key?
No. Bablix works with Apple's built-in translation engine at no cost. A Claude API key is optional and unlocks premium features: higher-quality translation, AI text refinement, and the "Hey Claude" assistant.
How do I get a Claude API key?
Visit console.anthropic.com, create an account, go to Settings → API Keys, and create a key. Add a payment method under Billing. Then paste the key into Bablix Settings. Typical conversation usage costs a few cents per session.
Why does the speech model take so long to download?
Bablix uses WhisperKit for on-device speech recognition. The model downloads once on first launch (~250-800 MB depending on your choice). After that, everything runs locally on your device.
Which languages are supported?
English, Chinese (Mandarin), Spanish, French, German, Japanese, Korean, Portuguese, Italian, and Arabic.
Can I use a Bluetooth remote?
Yes. Enable Bluetooth Remote in Settings. Most inexpensive Bluetooth shutter remotes work. Two remotes can be used — one per person — so each person controls their own translations without interrupting the other.
Why can't I adjust volume while using the app?
When Bluetooth Remote is enabled, the volume buttons become translation controls. Use Control Center (swipe down from top-right) to adjust volume, or disable Bluetooth Remote in Settings to restore normal volume buttons.
Is my data private?
Yes. Speech recognition runs entirely on your device. Transcripts are stored locally. No analytics, no tracking. Text is only sent to external servers when you use Claude-powered features (translation, refine, or Hey Claude). See our Privacy Policy for details.
The app says "API key required" for Refine
Refine and Hey Claude require a Claude API key even when using Apple Translation for the base translation. Add your key in Settings to enable these features.
